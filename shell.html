📋 Command Reference

□
×
Command Syntax Description Example
hlp hlp Show all available commands hlp
clr clr Clear the screen, reset cursor to top-left clr
sinf sinf Display system info (neofetch-style): OS, arch, kernel, FS, display, input sinf
room room List all files in root directory of FAT12. Shows name, extension, size in bytes. room
show show <filename> Read and print the contents of a file from FAT12 filesystem to screen. show HELLO.TXT
tuze tuze <filename> Open file in TUZE full-screen text editor. Creates file if it doesn't exist. Ctrl+S to save, Ctrl+Q to quit. tuze notes.txt
del del <filename> Delete a file from FAT12. Marks directory entry as 0xE5 and frees FAT clusters. del old.txt
mkf mkf <filename> [text] Create a new file. Optionally write text content immediately. If file exists, overwrites it. mkf hello.txt hi there
calc calc [expression] Calculator. Without args — interactive REPL mode (Ctrl+Q to quit). With args — evaluates immediately. calc 2**10 + 5
time time Show current time and date from RTC (CMOS). UTC+5 timezone. Format: HH:MM:SS DD/MM/YYYY. time
tez tez Enter TEZ language interactive REPL. Type one-line programs, press Enter to run. Ctrl+Q to quit. tez
run run <file.tez> Load a .tez source file from disk, compile to x86 and execute immediately. run prog.tez
