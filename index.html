# build
make clean && make
qemu-system-i386 -drive \
  file=os.img,format=raw,if=floppy
★ Features
_
□
×

🖥 Bootloader

Custom 16-bit bootloader with BPB/FAT12 header. Switches CPU from Real Mode to Protected Mode, loads kernel via BIOS INT 13h Extended Read (DAP).

⚙ JZA Kernel

32-bit flat memory kernel. IDT setup, PIC remapping, IRQ handlers for keyboard (IRQ1) and PIT timer (IRQ0). VGA text mode 80×25 with scroll support.

💾 FAT12 FS

Full FAT12 driver: read, write, delete files. ATA PIO disk driver. Root directory listing. Compatible with standard FAT12 floppy images (mtools).

⌨ Shell

Interactive shell with command history (↑/↓), backspace, command parsing. 10 built-in commands. Color output. Real-time clock in top-right corner.

📝 TUZE Editor

Full-screen text editor (inspired by nano). Line numbers, cursor navigation, Ctrl+S save, Ctrl+Q quit. Reads/writes files via FAT12.

🧮 TEZ Language

Custom programming language. Compiles to x86 machine code in-kernel. Variables, functions, if/else, while/for loops, cons.out() output.
